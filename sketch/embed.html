Embed TrussSketch

Add animated backgrounds to your website.

Copy & Paste

Add this to your HTML. A fullscreen animated background will appear.

<script src="https://cdn.trussc.org/sketch.js"></script> <script type="text/tcs"> float t = 0.0f; void draw() { t += getDeltaTime(); clear(0.05f); setColor(0.4f, 0.6f, 0.9f, 0.5f); drawCircle(getWindowWidth()/2 + cos(t)*100, getWindowHeight()/2 + sin(t)*100, 80); } </script>

Draw to Specific Canvas

Use data-canvas to target your own canvas element.

<canvas id="myCanvas" width="400" height="300"></canvas> <script src="https://cdn.trussc.org/sketch.js"></script> <script type="text/tcs" data-canvas="myCanvas"> void draw() { clear(0.1f); setColor(1.0f, 0.5f, 0.2f); drawCircle(getMouseX(), getMouseY(), 30); } </script>

Self-Hosting

Download files for version control or offline use.

1 Download these files:
sketch.js TrussSketch.js TrussSketch.wasm
2 Put them in the same folder, then:
<script src="./sketch.js"></script> <script type="text/tcs"> void draw() { // your code } </script>
