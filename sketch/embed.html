Embed TrussSketch

Add creative, interactive backgrounds to any website with just two lines of code.

Quick Start

Add this to your HTML:

<!-- Include TrussSketch --> <script src="https://cdn.trussc.org/sketch.js"></script> <!-- Write your sketch --> <script type="text/tcs"> void draw() { clear(0.1f); setColor(1.0f, 0.5f, 0.2f); drawCircle(getMouseX(), getMouseY(), 50.0f); } </script>

That's it! A fullscreen canvas is automatically created as a background.

Fullscreen Background

By default, TrussSketch creates a fullscreen canvas behind your content:

<body> <!-- Your content stays on top --> <div class="content"> <h1>My Website</h1> </div> <script src="https://cdn.trussc.org/sketch.js"></script> <script type="text/tcs"> float t = 0.0f; void draw() { t += getDeltaTime(); clear(0.05f); // Animated gradient circles for (int i = 0; i < 5; i++) { float x = getWindowWidth() / 2.0f + cos(t + i) * 100.0f; float y = getWindowHeight() / 2.0f + sin(t + i) * 100.0f; setColor(0.3f, 0.5f, 0.9f, 0.3f); drawCircle(x, y, 50.0f + i * 20.0f); } } </script> </body>

Custom Canvas

Target a specific canvas element:

<canvas id="myCanvas" width="400" height="300"></canvas> <script src="https://cdn.trussc.org/sketch.js"></script> <script type="text/tcs" data-canvas="myCanvas"> void draw() { clear(0.1f); drawCircle(200.0f, 150.0f, 50.0f); } </script>

Self-Hosting (Version Lock)

For production sites that need version stability, download and host the files yourself:

  1. Download these files:
    • sketch.js (loader)
    • TrussSketch.js (runtime)
    • TrussSketch.wasm (~2.8MB)
  2. Place them in your project:
    mysite/ assets/ sketch.js TrussSketch.js TrussSketch.wasm index.html
  3. Reference the local file:
    <script src="./assets/sketch.js"></script>

The loader automatically detects local hosting and uses local WASM files.
